# Rule-Based Chatbot

## Description
This project is a simple rule-based chatbot developed using Python.  
It responds to user inputs based on predefined rules using if-else
conditions and keyword matching.

## Features
- Responds to greetings
- Answers basic questions
- Handles unknown inputs
- Exit option for ending conversation

## Technologies Used
- Python

## How to Run
1. Open the folder in VS Code
2. Run the following command:
   python chatbot.py
3. Start chatting in the terminal
