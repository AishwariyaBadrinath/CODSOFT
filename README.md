# CODSOFT AI Internship Projects

This repository contains AI-based tasks completed as part of the **CODSOFT Artificial Intelligence Internship**.  
Each task focuses on implementing core AI concepts using Python.

---

## Tasks Overview

| Task No. | Project Title | Description | Key Concepts | Technologies |
|--------|---------------|-------------|--------------|--------------|
| Task 1 | Rule-Based Chatbot | A chatbot that responds to user inputs using predefined rules and keyword matching. | If-else logic, rule-based systems, basic NLP | Python |
| Task 2 | Tic-Tac-Toe AI | An AI agent that plays Tic-Tac-Toe against a human using optimal decision-making. | Minimax algorithm, game theory, search algorithms | Python |
| Task 3 | Recommendation System | A simple system that recommends items based on user preferences. | Content-based / collaborative filtering, similarity matching | Python, Pandas, Scikit-learn |
