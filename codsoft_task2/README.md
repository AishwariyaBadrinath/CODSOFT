# Task 2: Tic-Tac-Toe AI

## Description
This project implements a Tic-Tac-Toe game with an AI opponent using Python.  
The AI uses the Minimax algorithm to make optimal decisions and does not lose.

## Features
- Human vs AI gameplay
- AI uses Minimax algorithm
- Valid move checking
- Win and draw detection

## Technologies Used
- Python

## How to Run
1. Open the project folder in VS Code.
2. Run the following command:
   ```bash
   python tic_tac_toe_ai.py
