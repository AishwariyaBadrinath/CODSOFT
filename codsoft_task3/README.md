# Recommendation System

## Description
This project is a simple content-based recommendation system developed using Python.  
It recommends movies to users based on similarity of genres using TF-IDF and cosine similarity.

## Features
- Content-based filtering
- Uses cosine similarity
- Simple and beginner-friendly AI project

## Technologies Used
- Python
- Pandas
- Scikit-learn

## How to Run
1. Open the folder in VS Code
2. Install dependencies:
   pip install -r requirements.txt
3. Run the program:
   python recommender.py
4. Enter a movie name to get recommendations
